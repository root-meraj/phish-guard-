What is Phishing?

Phishing is a social engineering attack where attackers create fake websites that mimic legitimate ones to steal credentials, financial info, or personal data. Fake login pages are the most common vector.

BasicsSocial Engineering
🔍

Detection Techniques

  • URL Heuristics – Check for typosquatting, suspicious TLDs, excessive subdomains
  • Lexical Analysis – NLP to detect brand keywords in wrong domains
  • Visual Similarity – Computer vision to compare page layouts
  • Certificate Analysis – Check SSL issuer, validity period
MLNLP
⚠️

Common Red Flags

  • URL contains brand name but in wrong domain
  • HTTP instead of HTTPS
  • Unusual TLDs (.tk, .xyz, .ml, .ga)
  • IP address instead of domain name
  • Excessive subdomains
  • Recently registered domain (< 6 months)
  • Redirects through multiple URLs

How to Stay Safe

  • Always check the URL before entering credentials
  • Use a password manager (it won't autofill on fake sites)
  • Enable 2FA wherever possible
  • Hover over links before clicking
  • Use browser extensions like PhishGuard
  • Report suspicious sites to Google Safe Browsing
